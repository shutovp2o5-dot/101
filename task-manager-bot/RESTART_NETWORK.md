# Как применить настройки DNS в macOS

После добавления DNS серверов (8.8.8.8) в настройках сети, иногда нужно перезапустить сетевой интерфейс, чтобы изменения вступили в силу.

## Способ 1: Перезапуск сети через настройки

1. Откройте **Системные настройки** → **Сеть**
2. Выберите ваше подключение (модем)
3. Нажмите кнопку **"Отключить"** (если доступна)
4. Подождите несколько секунд
5. Нажмите **"Подключить"** или включите подключение снова
6. Подождите, пока подключение установится

## Способ 2: Перезапуск через терминал (требует пароль администратора)

```bash
# Найти имя вашего сетевого интерфейса
networksetup -listallnetworkservices

# Отключить и включить подключение (замените "ИмяПодключения" на реальное имя)
sudo ifconfig en0 down
sudo ifconfig en0 up
```

## Способ 3: Перезагрузка системы

Если ничего не помогает, можно перезагрузить Mac - это гарантированно применит все сетевые настройки.

## Проверка после перезапуска

После перезапуска сети проверьте DNS:
```bash
nslookup api.telegram.org 8.8.8.8
```

Если это работает, запустите бота:
```bash
cd task-manager-bot
python3 bot.py
```

## Альтернатива: Использование Wi-Fi

Если проблемы с модемом продолжаются, попробуйте:
- Подключиться к Wi-Fi сети
- Запустить бота через Wi-Fi
- После этого бот будет работать даже если переключитесь обратно на модем (если интернет работает)
